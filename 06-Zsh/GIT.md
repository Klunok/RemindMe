# These are common Git commands used in various situations:
p.s. "repo" is "repository"

## Usefull links:
```
https://git-scm.com/cheat-sheet
```
## Help information about help
```
git help
```
## Check the settings
```
git config --list
```

## Clone a repo into a new directory
```
git clone "link"
```

## Check the status
``` 
git status
```

## Create an empty repo 
```
git init
```

## Add file contents to the index
```
git add
```

## Add all files in repo
```
git add .
```

## Record changes to the repo
```
git commit
```

## Record changes and write comment
```
git commit -m "comment here"
```

## Update remote refs along with associated objects
```
git push
```

## Download the changes from repo
```
git pull
```
